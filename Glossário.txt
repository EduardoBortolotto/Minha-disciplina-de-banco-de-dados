Banco de Dados:
Um banco de dados é um sistema organizado para armazenar, gerenciar e manipular informações de forma estruturada. Ele permite que dados sejam armazenados em tabelas ou outros formatos, e facilita o acesso, a consulta, a inserção, a atualização e a exclusão de dados.
Essencialmente, um banco de dados é como um repositório onde as informações ficam guardadas de forma segura, mas de fácil acesso para quem tem permissão. Ele é usado em diversas aplicações, como sites, sistemas financeiros, aplicativos, entre outros.

SGBD:
Um SGBD (Sistema de Gerenciamento de Banco de Dados) é um software que facilita o processo de criação, gerenciamento, manipulação e controle de acesso a bancos de dados. Ele fornece as ferramentas necessárias para que usuários e programas possam interagir com o banco de dados de maneira eficiente e segura.
O principal objetivo de um SGBD é fornecer uma interface entre os usuários e o banco de dados, garantindo que as operações (como inserção, atualização, exclusão e consulta de dados) sejam feitas de forma organizada, segura e eficiente.

Diferença de BD e SGBD:
A diferença entre banco de dados (BD) e sistema de gerenciamento de banco de dados (SGBD) é que BD se refere ao conjunto de dados organizados, enquanto SGBD é o software ou a ferramenta que permite gerenciar, manipular e interagir com esses dados de forma eficiente.

Modelo relacional de BD:
O modelo relacional de banco de dados é um tipo de organização e estrutura de dados em que as informações são armazenadas em tabelas (ou relacionamentos), compostas por linhas (chamadas de tuplas) e colunas (chamadas de atributos). Esse modelo é baseado na teoria das relações matemáticas e foi proposto por Edgar F. Codd em 1970.

Oque é um microserviço?
Um microserviço é um estilo de arquitetura de software onde uma aplicação é estruturada como um conjunto de pequenos serviços independentes, que se comunicam entre si, geralmente através de APIs. Cada microserviço é responsável por uma funcionalidade específica da aplicação e pode ser desenvolvido, implantado, atualizado e escalado de forma independente.

APIs?
Uma API (Interface de Programação de Aplicações, do inglês Application Programming Interface) é um conjunto de regras e definições que permite que diferentes sistemas, aplicativos ou serviços se comuniquem entre si. Em outras palavras, uma API é uma interface que permite que um programa aceda funcionalidades ou dados de outro programa, sistema ou serviço, de forma padronizada.
